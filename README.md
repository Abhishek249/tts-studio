# 🎙️ TTS Studio

**Live:** https://abhishek249.github.io/tts-studio

Multi-character Text-to-Speech studio powered by ElevenLabs API.
Built for generating cinematic Hindi audio for Jain animated stories.

## Features
- Connect your ElevenLabs account
- Assign custom voices to characters (Narrator, Gautam, Dharini, Arishtanemi...)
- Write scripts using `[character]` and `[emotion]` tags
- Generate all segments in one click
- Play preview + download MP3

## Script syntax
```
[narrator] यह उस महान काल की बात है...

[gautam] [भावुक] हे प्रभो! मैं दीक्षा लेना चाहता हूँ।

[arishtanemi] [शांत] वत्स! जाओ। माता-पिता से आज्ञा लो।

[dharini] [रोते हुए] जाओ बेटा। मोक्ष-मार्ग पर जाओ।
```
